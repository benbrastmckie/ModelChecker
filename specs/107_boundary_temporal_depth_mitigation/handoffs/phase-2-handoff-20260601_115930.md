## Deviations from Plan

The plan described boundary-unsafe formula tests using formulas like G(G(p)) with insufficient M showing "spurious satisfaction" and sufficient M showing "correct behavior." Investigation revealed:

1. In BimodalSemantics, formulas like G(A) (no premises) are not simple temporal formulas -- they interact with world interval structure and abundance constraints. The solver behavior differs from standard temporal logic expectations.

2. Z3 marks many temporal formulas as UNSAT even at M < depth+2, because the bimodal frame constraints enforce non-trivial world structure that makes certain countermodels infeasible.

3. The plan's framing was adapted: instead of testing formulas that demonstrate boundary VACUITY (spurious SAT), the tests document the boundary claim mathematically (pure Python) and use known-good examples that demonstrate correct behavior at various M values.

This is consistent with the plan's own note: "This test documents the boundary effect rather than asserting a specific result, since the overall formula result depends on Z3's handling of vacuous universals."
